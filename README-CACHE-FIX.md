## ⚠️ 이미 예전 버전이 보인다면 (캐시 강제 초기화)

기존에 구버전을 열어봤다면 브라우저가 서비스 워커 캐시를 쥐고 있어 새 파일을 안 보여줍니다.
다음 중 하나로 해결:

**안드로이드 Chrome (가장 빠름):**
1. 홈 화면 PWA 아이콘 길게 눌러 **"앱 정보"** → 저장공간 → **"저장용량 삭제"**
2. 또는 Chrome → 사이트 주소 탭 → 🔒 → **"사이트 설정"** → 데이터 삭제
3. 다시 접속 → 새 v9이 로드됨

**PC Chrome:**
1. F12 개발자 도구 → **Application 탭** → **Service Workers** → Unregister
2. 같은 탭에서 **Storage** → **Clear site data**
3. Ctrl+Shift+R로 강제 새로고침

**설치된 PWA 앱:**
- 앱 삭제 후 홈 화면 추가 다시 진행

확인 방법: 타이틀 제목에 `[v9]` 표시, 시작 카드 하단에 `build v9` 텍스트,
콘솔에 `🫧 Bubble Dragons v9` 메시지 출력됨.

v9부터 서비스 워커가 **30초마다 업데이트 체크 + 자동 새로고침** 하도록 개선되어
앞으로는 이 문제가 재발하지 않습니다.
